Здесь описываются прогнозные модели.

Градиентный бустинг будет использоваться для первичного оценивания (без учёта прошлого PREMIUM).

Нейронная сеть будет использоваться для прогнозирования следующего значения (с учётом прошлого PREMIUM и длительности сотрудничества).

В соответствии с ноутбуками из eda - делим входные данные на две подвыборки по равенству нулю INSURED_VALUE.

Итого - **4 модели**, которыми руководит InsurancePremiumPredictor.

Прогнозирование с помощью NN производится с сохранением прошлых PREMIUM в *словарь-состояние*.

Прогнозы от NN обновляют *словарь-состояние*.

Есть возможность отправлять данные на инференс со столбцом PREMIUM, но принято решение в таком случае не обновлять с стояние, так как не ясна актульность этих данных (обучающие данные 3 года назад).

# Словарь-состояние:

(OBJECT_ID, flag) -> (прошлый PREMIUM, общая длительность сотрудничества)

flag - INSURED_VALUE == 0.
